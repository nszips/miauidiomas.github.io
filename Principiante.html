Saludos y presentaciones en nivel principiante

Hello = "HOLA"

También se puede escribir "Hi", "hey"



Good morning = "Buenos días"



Good Evening = "Buenas tardes" Este se usa para saludar.



Good Night = "Buenas noches" Este se usa para despedirse.



Welcome = "Bienvenido" Este se usa para recbir a una persona cuando llega a algun lugar, evento o grupo.
